# Resumen

En éste repositorio se almacena el software relacionado a la interfaz gráfica del proyecto final de semestre del equipo #1 en las materias de modelado matemático y programación. A su vez el repositorio contiene lo siguiente: 
- Código fuente para QT-Creator
- Diagramas de flujo de distintas partes del proyecto
- Instrucciones de uso

## Manual de uso
Dentro de la carpeta **INSTRUCCIONES**, revisar el archivo Instrucciones.md para ver de forma mas detallada el proceso de uso de la aplicación.

## Compilación
Al momento de escribir éste documento, la versión usada de QT Creator es la 18.2 community, con QT Framework 6.10.2. Para poder compilar y modificar la interfaz es necesario abrir 
el archivo CMakeLists.txt dentro del repositorio una vez se haya clonado completamente, y seleccionar los targets que se deseen tener dentro del IDE (Debug, release, etc.)
